## 一、创意来源

### 1.1 背景与痛点

在当今科研领域，研究人员面临着海量学术文献的信息过载问题。传统的文献阅读和综述撰写方式效率低下，难以应对快速增长的学术产出。研究人员需要一种智能化的工具来帮助他们快速理解和消化大量文献内容。

### 1.2 创新思路

灵析平台的核心创意来源于：

1. **多智能体协作**：借鉴人类学术讨论的方式，通过多个具有不同专业视角的AI智能体进行辩论式分析
2. **诺贝尔奖风格设计**：将AI生成内容提升到国际顶级期刊的视觉水准
3. **全流程自动化**：从文献检索到综述生成的一站式解决方案

### 1.3 核心价值主张

- **智能文献追踪**：7x24小时自动追踪全球最新学术动态
- **多视角分析**：多智能体辩论机制，突破个人认知局限
- **一键生成综述**：基于动态知识图谱的智能写作助手
- **诺贝尔奖风格排版**：国际顶级期刊级别的学术展示系统
- **团队协作画布**：可视化研究流程与知识管理

## 二、竞品调研

### 2.1 主要竞品分析

#### 2.1.1 Semantic Scholar

- **优势**：强大的语义搜索能力，丰富的文献元数据
- **劣势**：缺乏深度分析功能，无自动化综述生成

#### 2.1.2 Connected Papers

- **优势**：优秀的文献关联可视化
- **劣势**：功能单一，无写作辅助功能

#### 2.1.3 ResearchRabbit

- **优势**：文献推荐算法优秀
- **劣势**：界面简陋，无深度内容分析

#### 2.1.4 Elicit

- **优势**：问答式文献搜索体验好
- **劣势**：功能覆盖面窄，无完整工作流

### 2.2 灵析差异化优势

1. **多智能体辩论机制**：独创的支持者、批判者、方法论专家三方辩论分析
2. **诺贝尔奖风格设计**：业界首创的顶级学术视觉呈现
3. **全流程整合**：从检索到写作的完整科研工作流
4. **本地化部署**：支持多种部署方式，保护用户数据隐私

### 3.2 核心技术栈

#### 3.2.1 前端技术栈

- **框架**：Next.js 15 (App Router) + Gradio
- **语言**：TypeScript（严格模式）
- **UI组件**：Radix UI + Tailwind CSS
- **状态管理**：TanStack Query v5 + Zustand
- **PDF预览**：集成 Gradio PDF 组件

#### 3.2.2 后端技术栈

- **框架**：FastAPI + Uvicorn
- **语言**：Python 3.10+
- **PDF解析**：PyMuPDF (fitz) + Pdfminer.six
- **大模型集成**：智谱 GLM-4.5-Flash + MathTranslate
- **容器化**：Docker + Docker Compose

## 四、用户使用场景

### 4.1 核心用户群体

1. **高校研究人员**：需要快速了解领域前沿动态
2. **研究生**：撰写文献综述和毕业论文
3. **企业研发人员**：技术调研和竞争分析
4. **学术期刊编辑**：初筛投稿论文质量

### 4.2 典型使用场景

#### 4.2.1 场景一：研究新手入门

- **用户**：刚接触某研究领域的研究生
- **需求**：快速了解领域全貌和发展脉络
- **使用流程**：
    1. 输入研究主题关键词
    2. 选择"研究现状分析"模式
    3. 获取结构化综述报告
    4. 导出为Word/PDF格式

#### 4.2.2 场景二：专家深度研究

- **用户**：资深研究员
- **需求**：深入比较不同技术路线优劣
- **使用流程**：
    1. 输入具体技术问题
    2. 选择"方法比较分析"模式
    3. 查看多智能体辩论分析结果
    4. 获取关键洞察和研究建议

#### 4.2.3 场景三：团队协作研究

- **用户**：科研团队
- **需求**：多人协同完成大型综述
- **使用流程**：
    1. 创建研究项目
    2. 分配不同成员负责不同章节
    3. 实时共享分析结果
    4. 协作编辑最终报告

### 4.3 功能模块详解

#### 4.3.1 多类型文献综述

- **研究现状分析** (status)：全面了解领域发展现状
- **方法比较分析** (comparison)：深度比较不同技术方法
- **技术演进脉络** (timeline)：梳理技术发展历史
- **技术概念解析** (concept)：深入解析关键技术概念

#### 4.3.2 多智能体辩论机制

- **支持者视角**：挖掘文献中的支持证据和创新价值
- **批判者视角**：识别潜在问题和研究局限性
- **方法论视角**：评估研究方法的科学性和严谨性

#### 4.3.3 智能文献追踪

- 自动发现新发表的相关论文
- 动态更新知识图谱
- 邮件/消息推送通知

#### 4.3.4 论文格式修改系统

- **7类元素格式化**：论文标题、一/二级标题、正文段落、参考文献、表格、图片
- **学术字体库**：宋体、黑体、楷体、微软雅黑、Times New Roman
- **精细排版控制**：9种字号、5档行距、首行缩进、段前后间距
- **诺贝尔奖风格排版**：金色主题 + Playfair Display 字体的经典学术设计
