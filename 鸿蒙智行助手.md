# 鸿蒙智行助手 (HarmonyOS Travel Assistant)
基于HarmonyOS NEXT平台的AI出行助手，深度集成高德地图SDK和MCP Server，实现从出行规划到完美抵达的完整闭环体验。
## 🚀 项目特色
### 端侧创新
- **情景感知智能提醒**：基于日历事件和位置信息的主动出行服务
- **3D地图沉浸体验**：HarmonyOS NEXT 3D地图展示，支持室内地图和卫星图层
- **多模态路径规划**：支持步行、骑行、驾车、公交等多种出行方式
- **智能充电站搜索**：基于地理位置的充电站智能搜索和导航
### 云端能力
- **AI出行服务集成**：MCP Server提供12大出行服务
- **高德API聚合**：19项高德Web API服务统一管理
- **端云协同**：基于Cloud Foundation Kit的分布式数据同步

## 📋 环境要求
- **DevEco Studio**: 4.1 或以上版本
- **HarmonyOS SDK**: API 12 (5.0.0) 或以上
- **Node.js**: 18.0.0 或以上
- **高德地图API Key**: 需要在ApiConfig.ets中配置
## 🚀 快速开始
### 1. 克隆项目
`ash
git clone https://github.com/anbeime/gaode.git
cd gaode
`
### 2. 配置API Key
在 Application/entry/src/main/ets/config/ApiConfig.ets 中配置：
`	ypescript
export const AMAP_API_KEY = "your_amap_api_key_here";
`
### 3. 安装依赖
`ash
# 安装端侧依赖
ohpm install
# 安装云侧依赖
cd CloudProgram && npm install
`
### 4. 构建和预览
`ash
# 构建项目
ohpm run build
# 在DevEco Studio中预览
# 或使用命令行预览
ohpm run preview
`
