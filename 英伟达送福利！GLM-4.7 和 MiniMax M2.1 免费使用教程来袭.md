# 英伟达送福利！GLM-4.7 和 MiniMax M2.1 免费使用教程来袭

![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27400%27%20height=%27256%27/%3e)![image](https://p3-search.byteimg.com/img/labis/1df1c36d08accedf551a3a7bf6a89997~480x480.JPEG)

科技发展真快。人工智能新动态总是吸引人。尤其是那些喜欢探索新技术的朋友。每个新模型发布都让人兴奋。每一项福利活动都值得关注。2026 年 1 月 6 日，英伟达悄悄放出大福利。科技爱好者们都兴奋不已。

英伟达 NIM 开发者平台很神秘。它就像一个科技宝库。在大家不经意间，它上线了两个国产模型。智谱的 GLM-4.7 和 MiniMax M2.1。就像集市里出现稀世珍宝。摊主却没有大张旗鼓宣传。只是默默把宝贝摆了出来。虽然没有官宣，但 API 已经能用。最让人惊喜的是，这两个模型完全免费！

![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27400%27%20height=%27256%27/%3e)![image](https://p3-search.byteimg.com/img/pgc-image/1528360195298251b572d4a~480x480.webp)

有些朋友不了解 NIM 平台。简单说，它是个模型收纳盒。英伟达收集了主流开源模型。用统一 API 格式让大家调用。就像家里有个工具箱。Llama、Mistral、Gemma 这些模型都能找到。现在又多了国产 GLM 和 MiniMax。可真是越来越丰富了！

在这么多模型里，最推荐 GLM-4.7 和 MiniMax M2.1。GLM-4.7 是智谱 12 月底发布的。就像刚出炉的热面包。新鲜又美味。MiniMax M2.1 同一周发布。这两个模型像双胞胎兄弟。实力都不容小觑。

WebDev Leaderboard 排行榜上。MiniMax M2.1 以 1445 分排第六。GLM-4.7 以 1441 分排第七。它们的排名紧挨着。就像赛场上的两位劲敌。都在努力展现自己实力。英伟达把它们加入 NIM 平台。刚上线时资源可能抢手。服务偶尔不太稳定。token 输出速度有时较低。但对个人日常使用完全足够。这么好的福利，怎么享受呢？接下来详细讲使用教程。

![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27400%27%20height=%27256%27/%3e)![image](https://p3-search.byteimg.com/img/tos-cn-i-qvj2lq49k0/5a8ba467e3034b019fbdb32db11309ad~480x480.png)

既然要使用免费模型。首先得注册英伟达开发者账号。这一步像进入科技城堡。得先去门卫那里领通行证。具体怎么做呢？其实很简单。打开[build.nvidia.com](https://build.nvidia.com/)网站。就像来到城堡大门前。点击右上角 Login 按钮。就像按下了门铃。输入邮箱，告诉门卫我们是谁。之后设置密码。好比给通行证加安全锁。设置好密码点击 Create Account。就像告诉门卫准备好了。

这时邮箱会收到 6 位验证码。就像门卫给的特殊暗号。把这个验证码填进去。随便输入一个账户名。点击 Create NVIDIA Cloud Account。就成功迈进城堡第一步。

不过重点来了。进入页面后，右上角有 Verify 按钮。就像城堡里还有关卡要突破。点击这个按钮，输入手机号验证。这里告诉大家小秘密。亲测国内号码也可以。就像我们来自远方。城堡也同样欢迎我们。完成这一步，就注册好了。

接下来要获取 API Key。API Key 像打开宝藏房间的钥匙。没有它，没法使用免费模型。登录账号后，点击右上角头像。就像在城堡找到自己房间。然后选择 API Keys。接着找到 Generate API Key 选项。就像找到制作钥匙的机器。

给这个 Key 起个名字。好比给钥匙贴标签。方便我们识别。过期时间调成 "永不过期"。也就是 100 年。就像拥有长期使用的万能钥匙。点击 Generate Key 按钮。就会拿到长这样的 key：nvapi-xxx_xxx。这就是专属钥匙。一定要复制保存好。因为这个 Key 能调用 NIM 上所有免费模型。

![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27400%27%20height=%27256%27/%3e)![image](https://p3-search.byteimg.com/img/mosaic-legacy/2ed60003b21b3b2a25a3~480x480.jpeg)

拿到钥匙后，要接入 Cherry Studio。有些朋友不了解 Cherry Studio。没关系，可以看作者置顶文章。里面有详细介绍。点击左下角 "设置" 按钮。就像在 Cherry Studio 找到设置面板。然后选择 "模型服务"。再点击 "添加" 按钮。输入「老黄」，点 "确定"。

接下来配置连接信息。API 密钥这一项。粘贴刚刚生成的 API Key。就像把钥匙插入对应锁孔。API 地址这一项。填入[https://integrate.api.nvidia.com/v1](https://integrate.api.nvidia.com/v1)。就像告诉 Cherry Studio 要去哪里。模型这一项，推荐选择三个。z-ai/glm4.7。minimaxai/minimax-m2.1。moonshotai/kimi-k2-thinking。

配置完后，页面应该和教程图片一样。这说明已经成功连接。就像在两个世界间搭建桥梁。

完成以上所有步骤。就可以享受免费模型的乐趣了。经过实测，MiniMax M2.1 稳定性不错。对于轻度使用完全没问题。但要当作生产力工具来用。还是选择官方 Coding Plan 更好。

虽然需要花点钱。但它价格不贵，稳定可靠。速度也快，完全可以当作 Claude 平替。就像选择交通工具。免费公交车也能到达目的地。但想要更舒适快捷的出行体验。稍微花点钱坐出租车更好。
