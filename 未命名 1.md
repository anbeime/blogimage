📝 3️⃣ 踩坑总结：AI是“抽卡游戏”，多试有惊喜

<ol style="margin: 0 0 25px 0; padding-left: 20px; color: #333;">

<li style="margin-bottom: 12px; line-height: 1.6;">

**🎮 这是“抽卡游戏”不是标准答案**：同提示词刷5次每次效果不同，紫色→粉色→应援色白+蓝，多试就有惊喜。

</li>

<li style="margin-bottom: 12px; line-height: 1.6;">

**👀 给它“看”例子别只“说”想法**：说“粒子好看”没用，丢截图直接做发光粒子。

</li>

<li style="margin-bottom: 12px; line-height: 1.6;">

**🧩 分步骤别贪多**：一开始想同时加所有功能漏了鼠标缩放，拆成“模型+粒子→交互→卡片”每步可控更快。

</li>

<li style="margin-bottom: 12px; line-height: 1.6;">

**🎲 接受“不可复制”**：第二次卡片效果比第一次好但第三次变了，AI是随机灵感，意外效果可能更棒。

</li>

</ol>
