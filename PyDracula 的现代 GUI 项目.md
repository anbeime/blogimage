这个仓库是一个名为 **PyDracula** 的现代 GUI 项目，基于 PySide6 或 PyQt6 开发，提供了具有 Dracula 主题风格的用户界面模板。以下是该仓库的主要信息：

### 核心功能

- **现代主题支持**：包含深色和浅色两种 Dracula 风格主题（`themes/` 目录下的 `.qss` 文件），界面设计简洁且具有现代感。
- **自定义界面元素**：支持自定义标题栏、菜单切换、左右侧边栏切换等交互功能，提供灵活的布局调整。
- **高 DPI 适配**：通过设置环境变量 `QT_FONT_DPI` 缓解高分辨率屏幕下的显示问题。
- **可扩展性**：模块化结构设计，便于开发者添加自定义功能（如 modules/app_functions.py 用于扩展应用功能）。

### 项目结构

- **核心文件**：
    
    - main.py：应用程序入口，初始化界面并处理交互逻辑。
    - `main.ui`：Qt Designer 设计的界面文件，可通过 `pyside6-uic` 转换为 Python 代码（生成 ui_main.py）。
    - `resources.qrc`：资源文件（图片、图标等）配置，编译后生成 `resources_rc.py`。
    - setup.py：使用 `cx_Freeze` 配置，用于将应用打包为 Windows 可执行文件。
- **目录结构**：
    
    - `modules/`：包含界面功能（ui_functions.py）、应用设置（app_settings.py）等核心模块。
    - `widgets/`：自定义部件（如 `custom_grips/` 用于窗口大小调整）。
    - `themes/`：存储主题样式表文件。
    - `images/`：存放图片和图标资源。

### 使用与运行

1. **环境要求**：Python 3.9+ 和 PySide6（可通过 `pip install PySide6` 安装）。
2. **运行方式**：
    - Windows：`python main.py`
    - macOS/Linux：`python3 main.py`
3. **编译打包**：Windows 系统可通过 `python setup.py build` 生成可执行文件。
